กล้องสมาร์ตโฟน โดยไม่ต้องการอุปกรณ์เสริมพิเศษใดๆ
- LSTM (Long Short-Term Memory) ใช้สาหรับเรียนรู้ล าดับข้อมูลท่าทางมือในแต่ละเฟรมและ
จ าแนกภาษามือไทยเป็นค าศัพท์ได้อย่างแม่นย า
- Gesture Accuracy Detection Module ใช้ AI วิเคราะห์ความถูกต้องของท่าทางเทียบกับต้นแบบ
และให้ผลป้อนกลับในด้านมุมมือ ความเร็ว และต าแหน่งนิ้วแบบเวลาจริง

- Adaptive Learning Algorithm วิเคราะห์รูปแบบข้อผิดพลาดที่เกิดขึ้นซ ้าติดตามพัฒนาการระยะ
ยาวและปรับแผนการเรียนให้เหมาะสมกับระดับความสามารถของผู้ใช้แต่ละคนโดยอัตโนมัติ
- Speech Recognition ใช้สาหรับแปลงเสียงพูดของคู่สนทนาเป็นข้อความ รองรับทั้งภาษาไทยและ
ภาษาอังกฤษ

ตอบกลับที่เป็นธรรมชาติส าหรับระบบ Conversational AI
- TensorFlow / Keras ใช้สาหรับการสร้างและฝึกฝนโมเดล LSTM และโมดูล Gesture Accuracy
Detection
- TensorFlow Lite ใช้สาหรับแปลงโมเดลที่ฝึกแล้วให้เหมาะส าหรับการท างานบนอุปกรณ์มือถือ
- Speech Synthesis API (Text-to-Speech) ใช้ส าหรับแปลงข้อความภาษาไทยที่ ได้จากการ

- Array ใช้เก็บพิกัด (x, y, z) ของ Landmark ทั้ง 21 จุดของมือที่ตรวจจับได้ในแต่ละเฟรม
- Queue / Buffer ใช้เก็บล าดับเฟรมที่ผ่านมาเพื่อป้อนข้อมูลเข้าสู่โมเดล LSTM
- Object ใช้เก็บผลลัพธ์การแปล ประกอบด้วยคาศัพท์ภาษาไทยและค่าความเชื่อมั่น (Confidence

-Mapใช้จับคู่IDของท่าทางกับค าศัพท์ภาษาไทยและจับคู่รูปแบบข้อผิดพลาดกับค าแนะน าการ

Conversational AI

ค านวณคะแนนความถูกต้อง และระบุจุดที่ต้องปรับปรุง

ภาษาอังกฤษแบบเวลาจริง

ประโยคตอบกลับที่เป็นธรรมชาติ
- Auto BufferManagement Algorithm จัดการ buffer ของเฟรมให้มีขนาดเหมาะสมสาหรับการ
